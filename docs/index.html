ЧТО ТАКОЕ PALTOCRAFT?

PaltoCraft — это бесплатный неофициальный лаунчер для Minecraft, созданный для удобной игры. Он поддерживает официальную авторизацию через Microsoft, позволяет устанавливать любые версии игры, управлять модами и запускать игру в пару кликов.

Не требует пиратских аккаунтов — всё честно и безопасно.
